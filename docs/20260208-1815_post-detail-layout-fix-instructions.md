# 그림 피드 상세 페이지 수정사항 작업 지시서

## 목표
상세 페이지 레퍼런스 이미지 기준으로 상단 헤더를 최소화하고, 좌측 투표 레일을 제거하며, 최대 너비를 확대해 1920×1080 환경에서 정사각형 그림이 최대한 크게 보이도록 조정한다.

## 수정 범위
- 상세 페이지 헤더 구성 단순화
- 본문 레이아웃에서 좌측 화살표/숫자 제거
- 컨텐츠 최대 너비 확장 및 여백 재조정

## 비범위
- 실제 댓글 작성 기능, 투표/업보트 기능
- 서버 API 통합
- SEO/메타데이터 변경

## 적용 위치
- 라우트: `src/app/posts/[id]/page.tsx`
- 관련 컴포넌트 분리 시: `src/components/PostDetail.tsx`

## 수정 사항

### 1) 상단 헤더 단순화
- 상세 페이지에서는 **뒤로가기 버튼 + 로고만** 표시한다.
- 기존 TopNav의 다른 요소(탭, 버튼, 프로필 등)는 상세 페이지에서 제거한다.
- 헤더 높이는 기존 스타일 유지 가능 (`py-3`).

### 2) 좌측 투표 레일 제거
- 포스트 본문 좌측의 위/아래 화살표와 숫자 영역을 제거한다.
- 레이아웃은 단일 컬럼으로 재정렬한다.
- 본문 카드 내 컨텐츠는 좌측 정렬 유지.

### 3) 최대 너비 확대
- 최대 너비를 **더 크게** 확장하여 1920×1080 환경에서 정사각형 그림이 최대한 크게 보이도록 한다.
- 브라우저 URL 바와 헤더 높이를 고려해 메인 컨텐츠 영역의 가로 폭을 우선적으로 확대한다.
- 권장: `max-w-[960px]` ~ `max-w-[1100px]` 범위에서 시각적으로 가장 안정적인 값으로 조정.
- 카드 내부 여백은 `p-6` 유지, 필요 시 `p-8`까지 확대 가능.

## 구현 단계
1. 상세 페이지 전용 헤더를 구성하고, 상단에는 뒤로가기 + 로고만 노출.
2. 본문 레이아웃에서 좌측 투표 레일 제거 후 단일 컬럼으로 정리.
3. 컨텐츠 최대 너비 확장 및 1920×1080 기준 시각 확인.
4. 댓글 섹션 카드도 동일한 최대 너비 규칙 적용.

## 완료 기준
- 상세 페이지 상단 헤더가 뒤로가기 버튼과 로고만 표시됨.
- 좌측 화살표/숫자 레일이 완전히 제거됨.
- 1920×1080 화면에서 정사각형 그림이 이전보다 크게 보이고, 전체 레이아웃이 균형 있게 보임.
